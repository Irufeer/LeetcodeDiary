# LeetcodeDiary
A diary for Leetcode

## 说明
这个仓库目前只是我一个私人的记录解题过程的地方，暂时记录我在刷leetcode题目时的一些解法，还会记录我见到的，觉得很厉害的一些解法。
之后，我会逐步完善这个仓库，我的想法是，每一道题，记录解法从最简单到最难，复杂度逐步下降的多种解法，记录其中的思考过程，并扩展到多种语言，在解法中发挥语言优势。

## 记录信息
- 解法思想：暴力、贪心、动态规划等
- 复杂度：时间复杂度，空间复杂度，均摊复杂度等
